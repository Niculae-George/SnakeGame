This is my first game using raylib, a simple 2D game engine for c++.
# How to run
This repo already contains the compiled binary, so you can run it directly.
Also it has raylib already installed, so you don't need to install it.
# How to compile
To compile the game again after changing the code, you just need to save the cmake file and build it again.
